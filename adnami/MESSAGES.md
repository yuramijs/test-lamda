## RMB

### English 
```
Hi,

I’ve created and attached a script which contains the functionality that enables our Rich Media products for both direct and programmatic sales.

Notice that each website needs it's own unique script implemented.

The script must be implemented in such a way that it has been executed before any ad requests return from the ad server. Normally that can be achieved if it’s implemented directly in the <head> section of the website.

It can be loaded asynchronously but one will have to ensure load order manually.

Please let me know when it's implemented, so I can start the certification process.
```

### Danish 

```
Hej,

Jeg har oprettet og vedhæftet et script der indeholder den funktionalitet der gør vores Rich Media produkter tilgængelige for både direkte og programmatic salg.

Bemærk, at hvert medie skal have site eget unikke script implementeret.

Scriptet skal implementeres på en sådan måde, at det er afviklet inden nogen ad requests kommer tilbage fra ad serveren. Typisk kan dette opnås hvis det implementeres direkte i <head> sektionen af websitet.

Det kan indlæses asynkront, men man vil selv skulle sikre rækkefølgen af indlæsningerne.

Når det er implementeret melder I bare tilbage til mig, så vil jeg starte certificeringsprocessen.
```
